### Talo SDK for GameMaker

SDK для интеграции бэкенда Talo в проекты на GameMaker. Позволяет быстро подключить серверные функции (авторизация, хранение данных, аналитика и др.).

Импортируйте пакет в ваш проект GameMaker или добавьте исходные файлы вручную
Укажите ваш API ключ Talo в настройках SDK

Полная документация по API доступна в репозитории Talo: https://github.com/TaloDev
